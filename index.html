About

Saya merupakan seorang mahasiswa IT yang memiliki ketertarikan tinggi pada bidang web development terutama pada ranah backend developer. Selama satu tahun terahir saya aktif dalam mengembangkan kemampuan serta skill saya pada bidang backend developer dengan mengikuti berbagai macam kelas pembelajaran online mulai dari yang gratis hingga berbayar.

Saya memiliki pengetahuan mengenai bahasa pemrograman serta framework dalam membangun arsitektur backend seperti PHP, JavaScript, CodeIgniter, Laravel, NodeJs, Express, MySql dan MongoDB. Saat pernah mengikuti kegiatan Study Independen bersama Dicoding pada learning path pengembangan front-end web dan backend. Saya mengikuti kegiatan tersebut sebagai bentuk ketertarikan serta sarana dalam meningkatkan kemapuan saya mengenai web development.
